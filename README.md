# LockScreen
<p>LockScreen by NaiveHA</p>
<p>It is an Android app which locks the screen with a touch of a finger</p>
<p>The app makes use of the accessibility service: to work the user needs to enable it in:</p>
<p>Settings / Accessibility / Downloaded apps</p>
<p>The app does not require any permissions and does not collect any data</p>
<p>It is just a lock screen button within the reach of your thumb</p>
<p></p>
